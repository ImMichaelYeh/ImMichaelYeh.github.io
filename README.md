# Michael Yeh Portfolio

Personal portfolio website for Michael Yeh, a software engineer focused on backend systems and serverless AWS infrastructure.

**Live site:** [immichaelyeh.github.io](https://immichaelyeh.github.io/)

## Highlights

- Work experience at Optum and Cubic Corporation
- Education, technical skills, contact details, and downloadable résumé
- Links to selected projects:
  - [MyBudgeter](https://immichaelyeh.github.io/mybudgeter/)
  - [Target Archery Sight Calculator](https://immichaelyeh.github.io/sightCalculator/)

## Run locally

This is a static site. Open `index.html` in a browser, or serve the repository with any static file server:

```sh
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Structure

```text
index.html          Main portfolio page
css/style.css       Site styles
images/             Profile, education, and skill images
resumes/resume.pdf  Résumé
```
